Phoneme-Feature Analysis

Every sound ever produced by human vocal tract can be written as a vector of a finite number of values corresponding to the position of various articulators in the sound's production. Sounds tend to form classes based on values they have in common, and classes of sounds tend to behave alike when in contact with one another. As any spoken language has a finite inventory of sounds, the minimum set of values that suffices to uniquely describe any selected subset of segments can be easily computed. Likewise, given a set of values, a subset of segments with these values can be easily identified. This page serves as a tool to perform such analysis quicker and with more accuracy.
